# language-learning-app
Web-based English learning application with interactive flashcards, difficulty levels, and progress tracking

This is a simple web application for learning English words with translation into Ukrainian. The app allows users to choose a difficulty level and test their vocabulary knowledge.

## Features
- Three difficulty levels: Beginner, Intermediate, Advanced
- Random English words with Ukrainian translation
- User input validation and feedback
- Progress level calculation based on correct answers
- Animated word cards
- Responsive and modern UI design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery
